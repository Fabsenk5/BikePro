# Changelog

All notable changes to BikePro will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-03-04

### Added

- **⚙️ Dialed-In** — Suspension setup log with HSC/HSR/LSC/LSR, SAG%, travel, tire pressure tab (VR/HR)
- **🔧 Shred-Check** — Component wear tracker with 22 types, realistic service intervals, progress bars
- **🏔️ Park-Picker Pro** — Bikepark aggregator with 15 parks (DE/AT/CH/FR/IT), Go/No-Go scoring system
- **🎬 Trail-Video** — Curated YouTube trail video player with category filters
- **💨 Pressure-Bot** — Tire pressure calculator with tire type, casing, riding style, smart notes
- **📖 Ride-Log** — Ride journal with terrain type, difficulty (S0–S3+), descent, max speed, bike type
- **🔩 Component Tracker** — Bike master CRUD with 16 component types, dynamic setup fields
- **📚 Setup Guide** — MTB setup wiki with 34 articles across 7 categories
- **Dark theme** with neon accent colors per feature
- **Shared UI components** — BPCard, BPButton, BPSlider, BPPicker, BPModal, BPInput, BPProgressBar
- **AsyncStorage** persistence (local-first, Supabase-ready)
- **Vercel deployment** with automatic GitHub integration
- **Security audit** — 0 vulnerabilities, local-only data, no external API calls

### Planned

- 🚀 Jump-Analyzer — Airtime tracking and jump analysis
- 📍 Track & Trace — GPS route recording
- ☁️ Supabase cloud sync
- 📱 Native iOS/Android builds
